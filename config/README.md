# Config

Este directorio agrupa la configuracion del proyecto y la pagina de verificacion de GitHub Pages.

- `site/`: pagina estatica publicada por GitHub Pages.
- `site/fase-1/`: pagina independiente para Fase 1.
- `site/fase-2/`: pagina independiente para Fase 2.
- `package.json` y `tsconfig.json`: dependencias y configuracion heredadas del proyecto base.
- `src/`: codigo de apoyo que permanece disponible dentro de la carpeta de configuracion.
