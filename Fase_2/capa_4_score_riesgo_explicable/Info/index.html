Ruta prioritaria

Columna vertebral del MVP

Esta interfaz HTML es la ruta prioritaria de integración de la Fase 2. Los demás módulos deben entregar contracts, selected_contract, report, dashboard_metrics, red_flags, risk_patterns, connections y chatbot_context.

1. Abrir el HTML principal

El usuario entra por esta página única y ve el encabezado del proyecto con el nombre de Alejandro Montes.

2. Buscar y filtrar

Se usa la búsqueda principal y el filtro obligatorio de departamento para acotar el universo de contratos.

3. Seleccionar contrato

La tabla prioriza contratos con mayor riesgo, red flags, valor, proveedor repetido y baja pluralidad de oferentes.

4. Reporte y chatbot

La vista lateral construye el informe, el dashboard del contrato y el chatbot contextual solo con datos disponibles.
