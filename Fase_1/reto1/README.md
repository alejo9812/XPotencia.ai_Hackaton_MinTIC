# Reto 1

Este reto agrupa las dos bases de analisis de la Fase 1.

## Estructura

- `base1/`: primera base del reto 1, con su muestra, scripts y reportes.
- `base2/`: segunda base del reto 1, con su muestra, particiones Parquet y scripts de descarga.
- `tools/`: utilidades compartidas para descargar datos desde Socrata.

## Base 1

- Muestra CSV: `base1/meta/muestra_1000.csv`
- Muestra Parquet: `base1/meta/muestra_1000.parquet`
- Reporte EDA: `base1/reports/EDA_respuestas.html`
- Reporte de calidad: `base1/reports/calidad_datos.html`

## Base 2

- Muestra CSV: `base2/meta/muestra_1000.csv`
- Muestra Parquet: `base2/parquet/`
- Particiones mensuales: `base2/parquet/`
- Logs de descarga: `base2/meta/logs/`

## Nota

El objetivo es mantener todo el trabajo del reto 1 dentro de `Fase_1/reto1/` para que fase 2 no se mezcle con el análisis de datos.
