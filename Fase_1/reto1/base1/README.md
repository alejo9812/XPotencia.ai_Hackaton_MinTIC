# Reto 1 - Base 1

Carpeta organizada para conservar separados los entregables de la base 1.

## Contenido

- `eda_base1.py`: ejecuta el análisis EDA de la base 1.
- `calidad_datos_base1.py`: ejecuta el reporte de calidad de la base 1.

## Nota

Estos wrappers reutilizan los scripts raiz que ya existían en `Fase_1/`, para no duplicar lógica y mantener la separación por carpetas.
