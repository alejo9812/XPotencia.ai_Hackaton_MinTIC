Ruta tecnica

Repositorios y librerias utiles para pasar del borrador al MVP real.

  • pandas para leer CSV, Excel y tablas HTML.
  • pdfplumber y Camelot para extraer tablas desde PDF.
  • Scrapy para scraping web estructurado.
  • ydata-profiling y Evidently para analisis y reportes de calidad.
  • Metabase, Dash o Streamlit para dashboards y reportes compartibles.
